	VC and Delphi examples use the "MT_USBCamera_SDK.DLL" which is based on "cdecl" calling convention, for VB and C#
examples, they use "MT_USBCamera_SDK_Stdcall.dll", which is based on "stdcall" convention, for C# example, user might
have to copy the "MT_USBCamera_SDK_Stdcall.dll" and "MtUSBLib.dll" to "<WINDOWS>\system32" directory ( here, <WINDOWS>
is the windows installing directory, for it to run.
